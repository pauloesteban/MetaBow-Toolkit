# MetaBow Toolkit

## Requirements
- Max 8.1
- *MUBU* remote package

## Installation
1. Clone (or download) this repository
2. Move the `MetaBow-Toolkit` folder to `Max 8/Packages`
3. Install via Show Package Maganer
